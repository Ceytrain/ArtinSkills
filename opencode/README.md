# OpenCode 内置 Skills

来源：[anomalyco/opencode](https://github.com/anomalyco/opencode/tree/dev/.opencode/skills)

## 包含的 Skills

| Skill | 说明 | 安装路径 |
|-------|------|----------|
| android-native-dev | Android 原生开发 | `.config/opencode/skills/android-native-dev/` |
| ios-application-dev | iOS 应用开发 | `.config/opencode/skills/ios-application-dev/` |
| flutter-dev | Flutter 跨平台开发 | `.config/opencode/skills/flutter-dev/` |
| react-native-dev | React Native 跨平台开发 | `.config/opencode/skills/react-native-dev/` |
| frontend-dev | 全栈前端开发 | `.config/opencode/skills/frontend-dev/` |
| fullstack-dev | 全栈后端架构 | `.config/opencode/skills/fullstack-dev/` |
| shader-dev | GLSL 着色器开发 | `.config/opencode/skills/shader-dev/` |
| embedded-widget-dev | 嵌入式 Widget 开发 | `.config/opencode/skills/embedded-widget-dev/` |
| gif-sticker-maker | GIF 表情包生成 | `.config/opencode/skills/gif-sticker-maker/` |
| vision-analysis | 图片分析与 OCR | `.config/opencode/skills/vision-analysis/` |
| find-skills | Skill 发现与安装 | `.config/opencode/skills/find-skills/` |

## 安装

```bash
# 从 opencode 仓库复制到本地
cp -r <opencode-repo>/.opencode/skills/<skill-name> ~/.config/opencode/skills/
```
