# ArtinSkills

自定义 OpenCode Agent Skills 仓库。

## Skills

| Skill | 说明 |
|-------|------|
| html-report | HTML 报告生成（三色、双主题、侧边栏导航、Maple Mono NF CN 字体） |

## 安装

将 `html-report/` 目录复制到以下位置之一：

- **全局安装**：`%USERPROFILE%\.config\opencode\skills\html-report\`
- **项目级安装**：项目根目录 `.agents/skills/html-report/`
