# OpenCode 文档处理 Skills

来源：[anomalyco/opencode](https://github.com/anomalyco/opencode)

## 包含的 Skills

| Skill | 说明 | 安装路径 |
|-------|------|----------|
| docling | 文档处理与转换 | `.config/opencode/skills/docling/` |
| minimax-pdf | PDF 生成与设计 | `.config/opencode/skills/minimax-pdf/` |
| minimax-docx | Word 文档创建与编辑 | `.config/opencode/skills/minimax-docx/` |
| minimax-xlsx | Excel 电子表格处理 | `.config/opencode/skills/minimax-xlsx/` |
| pptx-generator | PowerPoint 演示文稿生成 | `.config/opencode/skills/pptx-generator/` |
| pdf-extraction | PDF 文本与表格提取 | `.agents/skills/pdf-extraction/` |

## 安装

```bash
cp -r <opencode-repo>/.config/opencode/skills/<skill-name> ~/.config/opencode/skills/
```
